### 📜 ATURAN PENTING KODE PMN (Agar AI Web Tidak Merusak Sistem Anda):

Pastikan AI web yang Anda gunakan mematuhi **Aturan Ketat** ini saat menyusun perbaikan:
1.  **Dilarang Mengedit index.html Secara Langsung**: Seluruh perubahan konten harus dilakukan pada file JSON modular (`data/parts/part_*.json`, `data/gl.json`, dll.) atau file sumber (`style.css`, `app.js`, `index.ui.html`), kemudian dikompilasi ulang.
2.  **Aturan escaping tanda kutip JSON HTML**: Karena teks HTML diletakkan di dalam string JSON, seluruh tanda kutip dua (`"`) di dalam elemen HTML **WAJIB** di-escape dengan garis miring terbalik (`\"`), contoh: `<p class=\"box-blue\">`.
3.  **Tautan Rujukan Silang (Xref Link)**: Semua tautan antar bagian naskah harus menggunakan format kelas dan ID data-sid yang presisi:
    `<a class=\"xref\" href=\"#1.1\" data-sid=\"1.1\">1.1</a>`
